
<div align="center">
  
  # NOTIFICATIONS SERVICE
</div>

<p>
O projeto é um sistema de serviço de notificações, desenvolvido durante o evento Ignite Lab 04 da RocketSeat
</p>

## Tecnologias

- TypeScript
- Node.js
- Nest.js
- Jest.js
- Prisma Studio
- SQLite


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
